# Survey Analysis Notebooks

This repository shares a small set of analysis notebooks from my thesis project in a format that can also be viewed by a non-coding audience.

The notebooks I will show are:

- one general analysis notebook: [survey_basic_analysis.ipynb](notebooks/survey_basic_analysis.ipynb)
- three group-specific notebooks: [survey_expert_analysis.ipynb](notebooks/survey_expert_analysis.ipynb), [survey_interested_analysis.ipynb](notebooks/survey_interested_analysis.ipynb), and [survey_peer_analysis.ipynb](notebooks/survey_peer_analysis.ipynb)
- one cross-group comparison notebook: [survey_peer_exp_comparison.ipynb](notebooks/survey_peer_exp_comparison.ipynb)

Rendered HTML versions can be opened directly in a browser where available.

The underlying survey data is not included because it is kept private under security and data protection guidelines.

Information about the Python environment used for the analysis can be provided on request.
